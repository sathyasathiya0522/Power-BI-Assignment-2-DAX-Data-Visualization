# 📊 Retail Sales Analysis Dashboard using Power BI

## 📌 Project Overview

This project demonstrates the complete data analysis process using Microsoft Power BI. It includes data modeling, calculated columns, DAX measures, interactive visualizations, and a dashboard to analyze retail sales performance.

The dashboard helps users understand sales trends, profit, order quantity, sales targets, and geographical sales distribution.

---

## 🎯 Project Objectives

- Build a Power BI data model
- Create calculated columns
- Develop DAX measures
- Design an interactive dashboard
- Generate business insights from retail sales data

---

## 🛠 Tools Used

- Microsoft Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)

---

## 📂 Dataset

The project contains the following tables:

- List of Orders
- Order Details
- Sales Target

---

## 🔗 Data Model

Relationships were created between the tables using appropriate keys to support dashboard analysis and DAX calculations.

---

## 📈 Calculated Columns

- Month
- Month Number
- Revenue Per Order
- Sales Category

---

## 📊 DAX Measures

- Total Sales
- Total Profit
- Total Quantity
- Order Count
- Profit Margin
- Total Target
- Average Profit (Delhi)
- YTD Sales

---

## 📉 Dashboard Visualizations

- Sales Target Achievement by Category
- Monthly Sales Trend
- Profit vs Quantity
- Order Count Analysis by State
- Geographic Sales Analysis
- Max Profit Margin by Sub-Category
- Total Sales KPI
- Total Target KPI
- Minimum Target KPI
- Sales Distribution by Sub-Category (Treemap)
- Sales Performance Matrix

---

## 💡 Key Insights

- Compared sales performance against targets.
- Identified monthly sales trends.
- Analyzed profit and quantity relationship.
- Evaluated state-wise order distribution.
- Visualized sales geographically.
- Identified profitable product sub-categories.
